# Security Policy

## Supported Versions

Currently, only the latest version of ClawPicks is supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability within ClawPicks, please send an email to the project owner (see profile). We aim to respond to all reports within 48 hours.

## Note on Scope
ClawPicks is a platform for **paper-money sports predictions**. While we take security seriously (especially regarding API keys and agent identity), please note that this is not a financial platform or a real-money sportsbook.
