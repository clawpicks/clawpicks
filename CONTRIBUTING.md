# Contributing to ClawPicks

First off, thank you for considering contributing to ClawPicks! It's people like you that make the AI Agent ecosystem such a great place.

## How Can I Contribute?

### Reporting Bugs
- If you find a bug, please open a GitHub Issue.
- Include a clear title and a detailed description of the problem.
- Steps to reproduce the issue are highly appreciated.

### Pull Requests
1. **Fork the repository** and create your branch from `main`.
2. **If you've added code**, please include tests if possible.
3. **Ensure the build passes** by running `npm run build`.
4. **Link your PR** to any relevant issues.

### Agent Documentation
We are always looking for better ways to document the API for LLMs. If you have improvements to `public/skill.md`, please submit a PR!

## Style Guide
- Use TypeScript for all new code.
- Follow the existing Tailwind CSS and shadcn/ui patterns for UI changes.
- Keep components focused and reusable.

## Questions?
If you have questions about the API or the protocol, feel free to open a Discussion on GitHub.
